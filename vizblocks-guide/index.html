Loading costumes/images

Before using VizBlocks, you should load costumes/images beforehand so that VizBlocks recognize that you want to use these assets in your visualizations.

Loading VizBlocks

Follow the procedure in the video to load VizBlocks.

Building a dot plot

In a dot plot, you need to read the key and the value (corresponds to the number of dots to display) before plotting the x-axis. The values are shown along the x-axis.

Building a pictograph

In a pictograph, you need to read the image(s) that you have loaded (refer to Loading costumes/images), the category name and its count value before drawing the pictures.

Building a bar chart

In a bar chart, you need to read the key (e.g category) and its corresponding value before plotting the x/y axis. The values are shown along the x/y axis.

Building a pie chart

In a pie chart, you need to read the key (e.g category) and its corresponding value. The values are shown in a pie with different colors.

Building a histogram

In a histogram, you need to read the key (e.g numerical value) and its corresponding value before plotting the x/y axis. The values are shown along the x/y axis.

Building a line chart

In a line chart, you need to read the x/y values before plotting the x/y axis. The values are shown along the x/y axis.

Building a scatter plot

In a scatter plot, you need to read the x/y values before plotting the x/y axis. The values are shown along the x/y axis. You can read in multiple values programmatically using a loop (refer to the video to see how this is done).

Building a heatmap

In a heatmap, you need to read the grid values of a 10x5 grid, then choose the image that you have loaded previously (refer to Loading costumes/images) to draw the grid on.

FAQ

1. My visualization is in a mess, how do I deal with it?

Use the “clear” block to clear the current screen of any leftover ink.

2. The visualization appears to be different from what I’ve expected, how do I fix it?

You might be using a combination of blocks that are not suitable for each other. Watch the videos of the respective visualization you are interested in creating for inspiration.
